# Brain Tumor Classification using CNN

A deep learning project for **brain tumor classification from MRI images** using **Convolutional Neural Networks (CNNs)**. This project focuses on applying computer vision techniques to assist medical image analysis and early diagnosis support systems.

---

## Project Overview

This model classifies MRI brain scans into tumor categories using a CNN-based architecture. The project includes data preprocessing, model training, evaluation, and prediction on unseen MRI images.

## Model Workflow

1. Load MRI image dataset
2. Preprocess and normalize images
3. Build CNN architecture
4. Train the model on labeled data
5. Evaluate performance on validation/test data
6. Predict tumor classes for new MRI images
